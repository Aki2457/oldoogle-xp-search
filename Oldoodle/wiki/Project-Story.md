# Project Story

Oldoodle XP started as a retro search page inspired by Windows XP, Internet Explorer, and old Google-style search pages.

Over the journey, it became a fuller project:

- A working search app.
- A fast hosted search backend.
- A public website.
- Browser extensions.
- A functional XP-style Start menu.
- Doodles for seasonal and joke themes.
- A tiny 404 desktop companion.
- Release packaging with guides.
- Docker self-hosting support.
- A speed-test website.
- Public documentation and explanation files.

The project stayed playful, but it also became practical: it can be opened, tested, shared, packaged, and self-hosted.

## What Makes It Different

Oldoodle is not just a normal search page. It is a tiny nostalgic desktop experience wrapped around modern web tooling.

It tries to make search feel:

- Fun.
- Old-school.
- Inspectable.
- Shareable.
- Easy to test.
- Easy to self-host.
