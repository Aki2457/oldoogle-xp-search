# Development Timeline

## Creation Phase

Oldoodle XP began as a Windows XP style search interface and grew through many iterations:

1. XP / Internet Explorer visual design.
2. Live search results.
3. Browser extension packaging.
4. Downloadable Chromium launcher.
5. GitHub repository and releases.
6. Doodles.
7. Functional Start menu.
8. Torrent release package.
9. GitHub Pages site.
10. Fast deployed search engine.
11. Speed-test site.
12. Docker self-hosting.
13. Browser toolbox widgets.
14. Icon and attribution page.
15. Public wiki.

## Official End

The creation journey officially ended at:

```text
11:23 PM, Singapore Time, 23 June 2026
```

That time is recorded here as the project’s public-ready milestone.
