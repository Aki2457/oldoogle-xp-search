# Features

## Oldoodle Search App

- Windows XP / Internet Explorer inspired interface.
- Old Google style search page.
- Live search widgets.
- Doodles: Classic, April Fools, Birthday, Winter, and Random.
- XP-style Start menu.
- 404 desktop companion.

## Search Engine

- Fast default search provider.
- Oldoodle-compatible JSON endpoint.
- Oldoodle-compatible live Server-Sent Events endpoint.
- Health endpoint for monitoring.
- CORS enabled for the hosted speed-test page.

## Browser Toolbox Widgets

The Oldoodle Chrome and Oldoodle Firefox extensions include a popup toolbox:

- Search.
- Timer.
- QR maker.
- Notes.
- Speed ping.

## Public Website

- GitHub Pages home page.
- Speed-test page.
- Made Possible With attribution page.

## Packaging

- Oldoodle Chrome ZIP.
- Oldoodle Firefox XPI.
- Windows app launcher.
- Release ZIP.
- Torrent metadata.
- Docker self-hosting files.
- Mac and Linux package scripts.
