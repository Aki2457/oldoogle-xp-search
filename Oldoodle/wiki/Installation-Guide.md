# Installation Guide

## Use The Public Website

Open:

```text
https://aki2457.github.io/
```

## Test Search Speed

Open:

```text
https://aki2457.github.io/speed-test/
```

The default endpoint is:

```text
http://localhost:3000
```

## Download The Release

Release page:

```text
https://github.com/Aki2457/oldoodle-search-engine/releases/tag/v1.0.3
```

The release ZIP contains the app package, extension packages, guides, and torrent metadata.

## Run Locally From Source

```sh
npm install
npm start
```

Then open:

```text
http://localhost:3000
```

## Build Packages

```sh
npm run package:extensions
npm run build:exe
npm run build:mac
npm run build:linux
npm run release:folder
```
