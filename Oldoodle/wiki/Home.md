# Oldoodle XP Wiki

Welcome to the public wiki for **Oldoodle XP**.

Oldoodle XP is a nostalgic Windows XP / Internet Explorer styled search project that grew into a complete shareable package: a public website, fast search backend, browser extensions, widgets, release zip, Docker self-hosting files, speed tester, and documentation.

## Official End Of Creation

The development journey officially ended at:

**11:23 PM, Singapore Time, 23 June 2026**

That timestamp marks the end of the creation phase and the beginning of Oldoodle as a public project.
